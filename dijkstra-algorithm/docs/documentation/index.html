drawGraph

Draw the full graph including optional step visualization and paths.

drawGraph(ctx: CanvasRenderingContext2D, viewport: Object, edges: Array, nodes: Array, positions: Map<number, {x: number, y: number}>, options: Object)
Parameters
ctx (CanvasRenderingContext2D)
viewport (Object)
Name Description
viewport.width number The width of the canvas.
viewport.height number The height of the canvas.
edges (Array) The array of edges in the graph.
nodes (Array) The array of nodes in the graph.
positions (Map<number, {x: number, y: number}>) The positions of the nodes.
options (Object) The options for drawing the graph.
Name Description
options.showSteps boolean (default false) Whether to visualize algorithm steps.
options.stepIndex number (default 0) The current step index to visualize.
options.dijkstraSteps Array (default []) The recorded steps of Dijkstra's algorithm.
options.astarSteps Array (default []) The recorded steps of A* algorithm.
options.dijkstraPath Array (default []) The final path found by Dijkstra's algorithm.
options.astarPath Array (default []) The final path found by A* algorithm.
options.mode string (default "both") Which algorithm to visualize ("dijkstra", "astar", or "both").
options.startId (number | null) (default null) The start node ID to highlight.
options.targetId (number | null) (default null) The target node ID to highlight.
